    百日誓师

    在上周星期五，我校怀新中学召开了百日誓师大会。

    百日誓师大会的流程为拍年级照——唱国歌——听领导讲话——教师和学生分别单独上台讲话——喊口号——学生和教师分别宣誓——走功门——在自班的横幅上签名——拍班级照——结束

    注意：上面从听领导讲话到学生和教师分别宣誓之间的顺序可能记错了。隔了几天，不是记得很清楚。

    在2 0 2 3 年 3 月 1 0 日的早上我班同学都穿着咱班红色的班服来到教室，教室里每天个人都穿着红色的班服，一片“红红火火”。

    上课铃响后，初三的学生们便集中好队排在了体育办公室上面的看台那排好队，（看台在学校的运动场上）。在我们排队时，运动场的入口处走进了一些举着彩旗或红旗的学生，他们不知道是初一的还是初二的。入口处又走来了几个穿着像国旗护卫队的服装的人。他们懒洋洋地走着。

    同学们说他们一点气势也没有。突然，那些国旗护卫队的人突然跑进了体育办公室，从里面拿出了一些礼炮，看来学校要为我们放炮呀！我班的同学们在看台上迅速地由红色蚂蚁变成了红色方阵，而隔壁的二班则是蓝色的卡片——他们也有班服，为蓝色。

    等着台上初三的同学们排好队后，老师们则排在了班级的前面。拍照那个人就在初三同学正前方隔着一条有八道的跑道处架了个梯子拍照。

    拍集体照拍了三次，前两次拍时，高铭同学故意举起右手，摆起纳粹的标志性手势。第三次拍照时可能迫于周围的言论，倒是没有再举起手了。

    拍完照后我们便到舞台前排好台，在篮球场，足球场，草地的上方队好队。像每次星期一升旗讲话一样，在开始前最要唱国歌。唱国歌时也像星期一一样全场静息息只有喇叭在那卖力地唱国歌。不过这种情况并没有保持多久。国歌唱到一半时，不知哪位大佬或者多位突然并成功地带动多我班唱国歌。声音哄亮，但全场似乎只有我们一个班唱，我班唱着唱着节奏还唱快了。

    随后就是千篇一律的领导发言了，发言后教师和学生分别派代表上台讲话。讲完后

    学生和教师们分别宣誓。宣誓的内容都很正常，积极向上。

    学生宣誓的内容

    （齐）我用青春的名义宣誓：

    面对挑战，我们毫不畏惧；面对困难，我们勇于向前；面对挫折，我们决不低头；面对失败，我们永不言弃；面对自己，我们不断超越。我发誓，

    敢于拼搏，用不懈争取进步；自强不息，用汗水浇灌理想；超越自我，用奋斗放飞希望；永不言弃，用信念实现梦想。我承诺，

    严守纪律，顽强拼搏，敢于竞争，超越自我。分秒必争，在此一搏！我们，注定成功！ 我们，必将辉煌！

    我们必将是怀新中学最棒的学子！

    宣誓完后，我们便每一个班都拿两个麦克风单独分别喊出自班的口号。我班的口号是“百日冲刺，一鼓作气，奋力前行，金榜题名”。我班是最后一个喊口号。因为是从两边的班开始喊口号而我班在中间。有的班是所有同学一起喊，有的班是几个人喊的。每个班的口号都不相同，有的班高喊“我要上高中”，引起了我班一些人的嘲笑，尽管“我要上高中”本身没有什么错。

    喊完口号后我们便要绕跑道顺时针走半圈然后走过成功门。我们第一个出发，二班随后。右手边为那些举着彩旗和红旗的学生们，他们不是初三的，旗手们站在跑道最内的那一圈的边缘，围跑道形成了一个圆。 我们跑道上排着队行走时，前方同学喊口号，前方的女生听到了跟着喊，但声音很小。而后方声音大的男生们还没听清前方队伍喊口号。前后方难以统一地喊口号。试了几次都不行，最后还是后方男生喊口号才能统一整齐地喊口号。

    经过2 0 0 米的路程，我们最终来到了成功门。成功门前的地上散落了一堆礼炮炸后的彩色碎片。而那些拿着礼炮的人则站了两队站在成功门的两边。

    我们排着整齐的队伍迈过了成功门，随即队伍就散，因为我们要与老师击掌。初三的老师在成功门后排成了两队，分边站在了路的两边。而教我们的老师大多在右边，所以我班同学基本上走右边与老师击掌。这里我要说，高铭同学的击掌与众不同，同学们大多是与老师击一次，而高铭同学则不停地与老师击掌，他一路走过去，每位老师可能与他去了四五次掌。与老师们击掌，每一次老师的手都是不同的。有的老师们手很柔软，而有的老师们手上则全是老茧。如与高成林化学老师击掌时，上一个老师的手是软的，而化学老师则一巴掌拍来。拍得我手疼，化学老师的手很硬，感觉他手上有一层厚厚的老茧。他是高铭的伯伯。

    与老师们击完掌后，我们一班和二班便走上了舞台。就只有我们一班和二班，因为我们是重点班，尽管老师们不再叫重点班了。我们来到舞台上，舞台上给领导坐的椅子被搬走了，桌上的红布也被拿走了，但那一排的桌子却留下了。老师们拿来了我班写有口号的横幅和几支黑色的记号笔。我们便在横幅上签下自己的名字。我写我的名字很大，在奋字偏左一点的地方上。在签字的同时，我们还与老师拍照，只要老师往那一站，就会有人上去与老师拍照。拍照时特别热闹，同学们纷纷与各位老师们拍照或与同学之间拍照。在拍照下过去了几十分钟，随后我们回到了成功门前拍照。在成功门前拍了几张班级照后，同学们又去忙着与老师拍照了。随后我们又上到了看台再次拍了张班级照，这次高铭没有举起纳粹标志性手势了。

    拍完这张班级照后，我们就要回教室继续上课了。回到教室里时，怡好下了第二节课。

    祝我中考胜利，金榜提名。

    那年在桂林旅行的伤心事

    今天看书，看到了一句话“只要老外往那儿一站，它就不再是偏远山区的房子，而是未来国际化的城市，”让我想起那年我家去桂林旅行。在路边的一个小吃店里吃东西时，旁边有一桌外国人在吃饭，我可以肯定他们是外国人，因为他们的肤色与我们的完全不一样，白的白，黑的黑。这些外国人吃了不一会儿就走了。但他们点了一桌子的菜还没吃多少，有的菜连一半的都没有吃完就走了。我在当时就对我的父母说他们浪费食物。

    但是我父亲却丝毫不给我面子就训斥了我，我当时感到特别地委屈，在家中，父母不是说要珍惜粮食吗？

    多年过去了，什么时候去桂林旅行我忘了。但在那个小吃店里的事我却从来没有忘记。当时我们吃的是羊肉饺子，吃的餐桌的位置我还记得。多年时间过去了，我还感到当时心中的委屈。
